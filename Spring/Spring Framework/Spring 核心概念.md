#### BeanFactory

底层支持两种对象模型

- 单例模型：提供具有特定名称的全局共享实例对象，Singleton 为默认模式，也是最常用的单例模型
- 原型模型：确保每次检索都会创建单独的实例对象，在每个用户都需要自己的对象时，采用原型模型

BeanFactory 是 Spring 实现 IOC 容器的基础
